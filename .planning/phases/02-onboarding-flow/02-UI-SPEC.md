# Phase 2 — UI Design Contract

> Visual and interaction contract for the onboarding flow: landing page, universe selection, and selfie upload modal. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | none (pure Tailwind utility classes) |
| Icon library | lucide-react |
| Font | Space Grotesk Variable (self-hosted via @fontsource-variable/space-grotesk) |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, tab gaps |
| md | 16px | Default element spacing, card inner padding |
| lg | 24px | Section padding, card gaps |
| xl | 32px | Layout gaps between sections |
| 2xl | 48px | Hero section vertical padding |
| 3xl | 64px | Page-level top/bottom spacing |

Exceptions: Touch target minimum 44px height for CTA buttons and genre tabs on mobile.

---

## Typography

| Role | Size | Weight | Line Height | Usage in Phase 2 |
|------|------|--------|-------------|-------------------|
| Body | 16px | 400 | 1.5 | Universe card premise text, primer screen body copy |
| Label | 14px | 400 | 1.4 | Genre tab labels, "Notify me" label, skip CTA, metadata |
| Heading | 20px | 600 | 1.3 | Universe card titles, primer screen headline "Make the story yours" |
| Display | 36px (mobile) / 48px (desktop) | 600 | 1.1 | Landing hero "Your face. Your story." |

Font family: `"Space Grotesk Variable"` (registered in `tailwind.config.js` as `font-sans`).

## Color

| Role | Token | Hex | Usage |
|------|-------|-----|-------|
| Dominant (60%) | `bg-base` | #0D0B12 | Page backgrounds, modal backdrop base |
| Secondary (30%) | `bg-surface` | #1A1724 | Universe cards, genre tab bar background, modal panel |
| Accent — Rose (10%) | `bg-rose-accent` / `text-rose-accent` | #D4799A | Primary CTA fill, genre tab active underline, "Start Your Story" button |
| Accent — Purple | `bg-purple-accent` / `text-purple-accent` | #9B7EC8 | Universe card gradient accent (romance genre), secondary visual interest |
| Accent — Gold | `text-gold` | #D4AF37 | Not used in Phase 2 (reserved for gem system in Phase 3) |
| Text primary | `text-text-primary` | #F0EAF8 | Headlines, card titles, CTA text |
| Text muted | `text-muted` | #8B7FA8 | Inactive genre tabs, card premise text, secondary body copy |

**Accent reserved for:** Primary CTA button fill, genre tab active underline, hover states on interactive cards. Never applied to body text, backgrounds at large, or decorative elements.

**Landing gradient background:** Linear gradient from `#0D0B12` (top) to `#1A0A1E` (bottom), applied as inline style or CSS class on the landing page wrapper.

## Component Inventory

### 1. Landing Page Hero

| Element | Specification |
|---------|---------------|
| Background | CSS gradient `#0D0B12` to `#1A0A1E`, full viewport height (`min-h-screen`) |
| Silhouettes | Inline SVG, `position: absolute`, bottom-aligned, `opacity: 0.3`, dark purple tones |
| Tagline | "Your face. Your story." — Display size, `text-text-primary`, centered |
| Sub-headline | Not specified in requirements — omit |
| CTA | "Start Your Story" — rose-accent filled pill, `px-8 py-3`, rounded-full, `text-base font-sans text-[#0D0B12]` |
| Layout | Flex column, centered vertically and horizontally, `z-10` above silhouettes |
| Container | `.page-container px-5` inner content |

**Hero animation sequence (Framer Motion):**

| Step | Element | Delay | Animation |
|------|---------|-------|-----------|
| 1 | Tagline | 0ms | `opacity: 0, y: 20` to `opacity: 1, y: 0`, duration 600ms |
| 2 | Headline | 200ms | `opacity: 0, y: 20` to `opacity: 1, y: 0`, duration 600ms |
| 3 | CTA button | 400ms | `opacity: 0, y: 20` to `opacity: 1, y: 0`, duration 600ms |

Easing: default spring or `ease-out`.

**CTA hover:** `scale: 1.05`, transition 150ms.

**CTA tap:** Routes to `/universes`.

## Accessibility Notes

- All interactive elements (CTA, tabs, cards) must have `focus-visible` ring styles
- Genre tabs: use `role="tablist"` + `role="tab"` + `aria-selected`
- Modal: `role="dialog"`, `aria-modal="true"`, `aria-labelledby` pointing to headline
- Modal trap focus while open; return focus to trigger element on close
- Skip CTA in selfie modal must be keyboard-accessible (not just visual text)
- Coming-soon cards: `aria-disabled="true"`, no click handler
- File input: `accept="image/*"` + `capture="user"`
