# Phase 3 — UI Design Contract

> Visual and interaction contract for the core reading loop: story reader, typewriter text reveal, choice mechanic, gem system, Your Story sidebar, and desktop 3-column layout. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | none (pure Tailwind utility classes) |
| Icon library | lucide-react |
| Font | Space Grotesk Variable (self-hosted via @fontsource-variable/space-grotesk) |
| Animation library | framer-motion |
| Bottom sheet library | vaul (install required) |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage in Phase 3 |
|-------|-------|-------------------|
| xs | 4px | Icon-to-text gaps (gem icon to count), inline padding |
| sm | 8px | Choice button internal icon gap, sidebar entry padding |
| md | 16px | Reading column paragraph spacing, choice button padding, sidebar section gaps |
| lg | 24px | Space between text block and choice list, sidebar heading margin |
| xl | 32px | Scene image bottom margin to reading column |
| 2xl | 48px | Reader page vertical padding (top/bottom safe area) |
| 3xl | 64px | Not used in Phase 3 |

Exceptions: Progress bar height is 2px (not on the 4px grid -- intentional for subtle visual). Touch targets on choice buttons minimum 44px height.

---

## Typography

| Role | Size | Weight | Line Height | Usage in Phase 3 |
|------|------|--------|-------------|-------------------|
| Body | 16px | 400 | 1.6 | Story prose text in reading column |
| Label | 14px | 400 | 1.4 | Gem count, sidebar decision entries, chapter title in nav, choice gem badge text |
| Heading | 20px | 600 | 1.3 | "Your Story" sidebar heading, gem gate sheet heading |
| Display | 28px | 600 | 1.2 | Chapter title overlay on scene image |

Font family: `"Space Grotesk Variable"` (registered in `tailwind.config.js` as `font-sans`).

Note: Story prose body uses 1.6 line-height (not the Phase 2 standard 1.5) for improved readability on long-form text. This is the only exception.

## Color

| Role | Token | Hex | Usage in Phase 3 |
|------|-------|-----|-------------------|
| Dominant (60%) | `bg-base` | #0D0B12 | Reader page background, sidebar background |
| Secondary (30%) | `bg-surface` | #1A1724 | Choice buttons default background, sidebar panel background on desktop, gem gate sheet background, nav bar background |
| Accent -- Rose (10%) | `bg-rose-accent` / `text-rose-accent` | #D4799A | Progress bar fill, selected choice gradient (left edge), choice hover border, sidebar decision dot indicator |
| Accent -- Purple | `text-purple-accent` | #9B7EC8 | Not primary in Phase 3 -- available for scene image gradient overlay accent |
| Accent -- Gold | `text-gold` / `bg-gold` | #D4AF37 | Gem counter icon + number, gem badge pill on locked choices, gem gate sheet icon |
| Text primary | `text-text-primary` | #F0EAF8 | Story prose, choice button text, sidebar heading, nav chapter title |
| Text muted | `text-muted` | #8B7FA8 | Unchosen choice text (at 30% parent opacity), sidebar timestamps, gem gate body copy |

**Accent reserved for:**
- Rose: Progress bar fill, selected choice left-border gradient, sidebar decision dot, choice button hover border
- Gold: Gem counter display, gem badge pill on locked choices, gem gate sheet CTA icon. Never applied to body text or non-gem UI.

**Selected choice gradient:** `bg-gradient-to-r from-rose-accent/20 to-transparent` on `bg-surface` base.

**Unchosen choice dimming:** Parent container `opacity-30` on unchosen choices after selection is made.

### 7. Story Text Area (READ-02, READ-03)

| Property | Value |
|----------|-------|
| Container | Within reading column, `py-8` |
| Text style | Body (16px), weight 400, `text-text-primary`, line-height 1.6 |
| Paragraph spacing | `mb-4` between paragraphs (16px) |
| `[Name]` rendering | Replace with stored user name from Zustand, same styling as surrounding text |
| Tap target | Entire text container area is tappable (`onClick` / `onTouchEnd`) |
| Cursor | `cursor-pointer` on text area during typewriter animation |

**Typewriter animation (useTypewriter hook):**

| Property | Value |
|----------|-------|
| Method | Recursive `setTimeout` with `isCancelledRef` (not setInterval) |
| Speed | 30ms per character default |
| Cancel guard | `useRef<boolean>` -- set `true` on unmount and on skip-tap |
| Output | Returns `{ displayedText, isComplete, skip }` |
| Skip (first tap) | Sets `isCancelledRef.current = true`, immediately sets full text |
| Advance (second tap) | Only fires when `isComplete === true`, calls parent `onAdvance` callback |

## Responsive Behavior

| Breakpoint | Layout Change |
|------------|---------------|
| < 1024px (mobile/tablet) | Single column: scene image full-width at top -> sticky nav -> reading column centered (max-w-680px, px-5) -> choices. Sidebar via Vaul bottom sheet. |
| >= 1024px (lg) | 3-column grid: 280px fixed sidebar (left) + 680px reading column (center) + remaining space. Scene image full-width above grid. Sidebar always visible. Nav bar "Your Story" trigger hidden. |
| >= 1440px | Content capped by grid column widths. No `.page-container` needed -- grid handles it. |

## Accessibility Notes

- Choice buttons: `role="button"`, keyboard accessible via `Enter`/`Space`
- Selected choice: `aria-pressed="true"`
- Locked choice: `aria-label` includes gem cost, e.g. "Choose: {text} (costs 10 gems)"
- Gem gate sheet: `role="dialog"`, `aria-modal="true"`, `aria-labelledby` pointing to heading
- Sidebar: `role="complementary"`, `aria-label="Your Story sidebar"`
- Mobile sidebar: `role="dialog"` when rendered as Vaul sheet
- Progress bar: `role="progressbar"`, `aria-valuenow`, `aria-valuemin="0"`, `aria-valuemax="100"`
- Loading shimmer: `aria-busy="true"`, `aria-label="Loading story content"`
- Typewriter text: `aria-live="polite"` on text container (screen readers get final text, not character-by-character)
- Tap-to-skip: keyboard `Enter` or `Space` on text area also triggers skip/advance
